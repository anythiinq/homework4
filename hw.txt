- Explain in a sentence or two why the call to `Map<Coord, int>::insert` causes at least one compilation error.
    
    When Map inserts a new element, it keeps itself in a sorted order. Because the class `Coord` doesn’t define its own `<` definition, `insert` won’t know which order to insert `Coord`, causing a compilation error

# question 6

- code
    
    ```cpp
    void reassign(const Map& m, Map& result)
    {
          // Guard against the case that result is an alias for m (i.e., that
          // result is a reference to the same map that m refers to) by building
          // the answer in a local variable res.  When done, swap res with result;
          // the old value of result (now in res) will be destroyed when res is
          // destroyed.
    
        Map res;
    
        if (!m.empty())
        {
            KeyType prevKey;
            ValueType value0;
    
              // Get pair 0, which must succeed since m is not empty
    
            m.get(0, prevKey, value0);
    
              // For each pair i after pair 0, insert into res a pair with
              // pair i-1's key and pair i's value.  (This loop executes 0 times
              // if m has only one pair.)
    
            for (int i = 1; i < m.size(); i++)
            {
                KeyType k;
                ValueType v;
                m.get(i, k, v);
                res.insert(prevKey, v);
                prevKey = k;
            }
    
              // Insert a final pair with last pair's key and pair 0's value.
    
            res.insert(prevKey, value0);
        }
    
        result.swap(res);
    }
    ```
    
- part a) what is the time complexity?
    
    The time complexity is $O(N^2)$. `get` outside the loop will execute $\frac{N}{2}$ times. Inside the loop, `findFirstAtLeast` visits all the nodes until it finds where key is. The result there is $O(N)$ for one iteration. The `for` loop of `reassign` iterates $N-1$ times, so the result is $N(N-1)$. The last insert adds $O(N)$, and `swap` is $O(1)$. Therefore, the highest time complexity must be $N^2$.
